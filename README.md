# Tempus Copilot Case Study
Implementation of the Tempus Case Study for the GenAI Product Builder role, completed by Jacob Slabosz.

The backend is deployed to [Render](https://tempus-copilot-backend.onrender.com). Check the site's status by visiting [https://tempus-copilot-backend.onrender.com/health](https://tempus-copilot-backend.onrender.com/health)

To run the code locally on a Mac, run the following in a terminal:

```
./scripts/setup.sh
```
```
./scripts/run.sh
```

The backend application will begin running on localhost port 5001.
